## 🚀 Overview

BookNest is a **full-stack web application** that allows users to manage and explore their personal book collections.

It includes:
- Secure user authentication  
- Cloud-based storage for books and PDFs  
- Real-time book management features  
- Clean and responsive user interface  

---

## ✨ Key Features

- 🔐 **Authentication** — Login & Signup using Firebase Auth  
- 📚 **Book Management** — Add, view, and manage personal book collections  
- 📄 **PDF Viewer** — Integrated PDF rendering for reading books  
- 📱 **Responsive Design** — Works smoothly across devices  

---

## 🛠️ Tech Stack

**Frontend:** React.js, Vite, React Router  
**Styling:** CSS Modules, Vanilla CSS  
**Backend & Database:** Firebase (Auth, Firestore, Storage)  
**Utilities:** Axios, React-PDF  

---

## 🏗️ Architecture

- **Frontend (React):** Handles UI, routing, and state management  
- **Firebase Auth:** Manages user authentication securely  
- **Firestore:** Stores book metadata  
- **Cloud Storage:** Stores uploaded files (PDFs/images)
